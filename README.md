# Overview

## Human Interface Guidelines

이 문서는 Apple Developer에서 제공하는 Human Interface Guidelines 공식 문서를 활용합니다. Apple 플랫폼의 서비스를 개발하는 한국 개발자의 이해를 돕기 위해 영문에서 한국어 번역을 진행하고 있습니다.

휴먼 인터페이스 지침에는 모든 Apple 플랫폼에서 훌륭한 사용자 경험을 설계하는 데 도움이 되는 지침과 모범 사례가 포함되어 있습니다.

휴먼 인터페이스 지침을 통해 Apple 플랫폼과 원활하게 통합되는 훌륭한 앱을 설계하기 위한 심층적인 정보와 Ul 리소스를 얻을 수 있습니다. 휴먼 인터페이스 지침은 더 효율적으로 작업하고 애플리케이션을 더 응집력 있게 만드는 데 도움이 되는 구성 요소와 도구를 제공합니다.
